LLMs
====

LLMs are the core of **Task Planner** and **Response Generator** components. LLMs are used by the **Task Planner** to understand user queries, break it into **Tasks** needed 
to be executed. They are also used by **Response Generator** to provide the final answer to the user. 
LLMs can also be used in **Tasks** for example, using it for user emiotion assessment.
